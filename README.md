# ポケモン チャンピョンズ 内定メガシンカ変化データベース

『ポケモン チャンピョンズ』に内定しているポケモンの、メガシンカ前後のステータス変化を確認・比較できるWebツールです。

## 🚀 サイトURL

[https://taketake-dev.github.io/pokemon-champions-stats/](https://taketake-dev.github.io/pokemon-champions-stats/)

## ✨ 主な機能

- **種族値の動的差分計算**: 通常時とメガシンカ後の種族値を比較し、上昇・下降量を表示します。
- **リアルタイム検索**: ポケモン名を入力すると、該当するポケモンを即座にフィルタリングします。
- **ひらがな入力対応**: `ふしぎ` などのひらがな入力でも `フシギバナ` などの検索候補が表示されます。
- **視認性の高いカードデザイン**: メガシンカ前後の情報を整理し、対戦中も素早く確認できるUIを採用しました。
- **レスポンシブ対応**: PCだけでなく、スマートフォンからの閲覧にも最適化しています。

## 🛠️ 使用技術

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Data Management**: JSON (外部ファイル化によるデータとロジックの分離)
- **Tool**: Google Analytics
- **Deployment**: GitHub Pages

## 💡 開発のこだわり

- **データの正規化**: タイプのリスト化や差分の自動計算ロジックを実装することで、データ管理の効率化と表示の正確性を両立させました。
- **就職活動に向けた設計**: 単なるファンサイトではなく、保守性やユーザーの利便性を考慮した「Webツール」としての設計を意識しました。
